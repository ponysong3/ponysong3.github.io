Python程序结构

Posted on 2021-01-02 Edited on 2022-07-02 In Python

上学期间写作文会先列个提纲，目的就是为了使作文有个好的结构。纵使平时积累再多的词汇、语法、素材，等到了列提纲的时候也得简单、清晰、明了。类似地，Python的三方库和方法纷繁复杂，但它自身的语法要素和程序结构却简单、清晰、明了。语法要素已经在“Python语法元素分析”一文中介绍过了，本文将介绍Python程序控制的方法论，即程序结构和语法。

Python程序结构包括顺序结构、分支结构、循环结构。顺序结构无需赘述，着重介绍分支结构和循环结构。

除了操作符，Python也提供了三个用来对条件进行组合的保留字。

在Python语言中，只提供两种循环方式，一种叫遍历循环，对应保留字for；另一种叫无限循环，对应保留字while. 掌握这两周循环就掌握了Python所有的循环表达方式。

循环控制保留字

在所有的编程语言中，break和 continue都是循环控制的保留字。

  • break跳出并结束当前循环，执行循环后的语句。即打破整个循环，如果是多层循环，则打破当前层。
  • continue结束当次循环，继续执行后续次数的循环。即打破当次循环。

他山之石：random库

random库是用于生成随机数的Python标准库，即随着Python解释器而自带的库，不需要安装。

严格来讲，计算机产生的只是伪随机数，即在特定算法规则下产生的数字，这种数字超出了常人的预测能力，所以我们可以把它当成“随机数”. 计算机通常采用梅森旋转算法生成伪随机序列，这个序列中的每个元素称伪随机数。基本原理是：随机数种子---梅森旋转算法---随机序列。只要种子相同，随机序列和序列中元素就是确定的

基本随机函数

函数 描述
seed(a=None) 初始化给定的随机种子，默认为当前系统时间
>>>random.seed(10) #产生种子10对应的随机序列
random() 生成一个[0.0, 1.0)之间的随机小数
>>>random.random()
0.5714025946899135这是种子10对应随机序列中的第一个元素

程序中使用随机数种子是因为有时候需要复现程序运行结果，如果不指定种子，随机数默认使用系统时间作为种子，而系统时间是精确到微秒的，很难复现

扩展随机数函数

函数 描述
randint(a, b) 生成一个[a, b]之间的整数随机数
>>>random.randint(1, 999)
357
randrange(m, n [,k]) 生成一个[m, n] 之间以k为步长的随机整数
>>>random.randrange(1, 999, 10)
251
getrandbits(k) 生成一个k比特长的随机整数
>>>random.getrandbits(5)
15
uniform(a, b) 生成一个[a, b]之间的随机小数随机
>>>random.uniform(1,8)
7.884477557916984
choice(seq) 从序列seq中随机选择一个元素
random.choice([1,2,3,4,5,6,7,8,9,0])
6
shuffle(seq) 将序列seq中元素随机排列，返回打乱后的序列
>>>s=[1,2,3,4,5,6,7]; random.shuffle(s);print(s)
[2, 4, 6, 7, 3, 5, 1]
