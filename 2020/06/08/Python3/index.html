海龟绘图

Posted on 2020-06-08 Edited on 2020-10-12 In Python

学习和认知的过程也遵循“看山是山，看山不是山，看山还是山”的真谛。当然，真谛需要正确的打开方法，尤其对于Python上百万的编程库来说，迫切地需要高效的学习方法。下面通过海龟库的学习，探索一下学习路径“感受代码——理论基础——语法元素——举一反三”

感受代码

理论基础

turtule库基本介绍

  • turtle(海龟)库是turtle绘图体系的Python实现
  • turtle绘图体系诞生于1969年，主要用于程序设计入门，就像windows的画图板
  • 想象一只海龟在画布上游走，走过的轨迹就形成了绘制的图形，海龟可以改变颜色、宽度等
  • turtle库是标准库
    • 标准库：随解释器直接安装到操作系统中的功能模块
    • 第三方库：需要经过安装才能使用的功能模块

turtle绘图窗体布局

  • turtule绘图窗体即屏幕上的一块画布空间，尺寸的最小单位是像素
  • 屏幕左上角是屏幕坐标系的原点(0,0)
  • 设置turtule绘图窗体的语句：turtle.setpup(width,hight,startx,starty)
    • width,hight控制窗体的宽度和高度
    • startx,starty控制窗体左上角相对屏幕左上角的位置
    • starts,starty可选
    • setup()不是必须的

turtle空间坐标系

  • 绝对坐标: turtle绘图窗体的中心窗体坐标系的原点(0,0)，相对该原点的坐标就是绝对坐标, 相关函数turtle.goto(x,y)
  • 海龟坐标: 将“海龟”当前位置作为原点(0,0),相对“海龟”当前位置的坐标就是海龟坐标, 相关函数turtle.fd(d), turtle.bd(d), turtle.circle(r,angle)

turtle角度坐标系

  • 绝对角度：类同绝对坐标, 相关函数turtle.seth(angle)
  • 海龟角度：类同海龟坐标, 相关函数turtle.left(angle), turtle.right(angle)

RGB色彩体系

  • RGB三个通道上取值0-255或0-1组成万物色
  • turtle默认使用0-255整数值表示颜色
    • turtle.colormode(1.0)切换到RGB小数值模式
    • turtle.colormode(255)切换到整数值模式

语法元素分析

库引用

  • 扩充Python程序功能的方式

  • 使用保留字import完成，采用 <a>.<b>( ) 编码风格, 具体表现为

    • import Library
    • from Library import *
    • import Library as L

turtle画笔控制函数

penup(), pendown():画笔操作后一直有效，一般成对出现。“抬起画笔, 海龟在飞行, 放下画笔, 海龟在爬行“
pensize(): “画笔大小, 海龟的腰围“
pencolor():画笔设置后一直有效. ”画笔颜色, 海龟的涂装”. pencolor(“purple”) 或 purple(r,g,b)

turtle运动控制函数

fd(d): “海龟向前进，海龟走直线”. 距离d可以是负数, 代表后退
circle(r, angle): “海龟绕着左侧的圆心画圈”. 默认画整圆. 半径可以是负数, 代表圆心在右侧

turtle方向控制函数

seth(): “海龟面向的绝对角度”
left(), right(): “海龟视角旋转的角度”
