有图有数据: Python数据分析

Posted on 2020-06-23 Edited on 2020-10-12 In Python

公认的Python学习路径是：爬虫–数据分析–(机器学习)人工智能。的确这样，Python爬虫作为学习Python的入门知识，具有十足的吸引力和超强的获得感。而渐渐爬取大量数据之后，需求驱动前往数据分析的知识领域进军

工善利器

工欲善其事，必先利其器。ANACONDA是学习Python尤其数据分析首推的软件，该软件集成了Python、Conda、Jupyter等，即安装了ANACONDA就不需要单独安装Python了，同时Conda是一个第三方包管理软件，可以通过Conda安装Numpy、Matplotlib、Pandas库，这些库是数据分析必备的。Jupyter是大众推荐的、执行数据分析操作的、基于Web的交互软件。以上就是进行数据分析需要搭建的Pyton环境了

师出有名

Numpy、Matplotlib和Pandas分别是做什么的？它们在数据分析中的作用是什么？长话短说，

  • Numpy是用来组织数据的，就像列表用来组织有序列的数据一样，Numpy可以更有效地组织N维数据，为数据处理提供了结构化基础。嗯，很晦涩，但没关系，知道它是必须的就行了
  • Matplotlib是用来展示数据的，说白了就是根据数据绘图，这个就好理解了，这是我们数据分析的重要目的
  • Pandas在Numpy的基础上进一步组织数据，并利用Matplotlib方法库来进行绘图，可以简单的理解为Pandas整合了Numpy和Matplotlib的方法
