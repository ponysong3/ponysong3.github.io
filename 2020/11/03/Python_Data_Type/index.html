Python数据类型

Posted on 2020-11-03 Edited on 2022-07-02 In Python

重点介绍两种基本数据类型：数字类型和字符串类型。对于每种类型，分别介绍其表示方式和操作方式。比如，数字可以表示成二进制、八进制、十六进制形式，字符串可以用单引号、双引号、三引号表示；数字可以通过操作符或操作函数进行运算，字符串可以通过操作符、操作函数、对象方法进行编辑。更普遍地，任何其他数据类型都有其“表示方式”和“操作方式”两方面。

数字类型与操作

三种数字类型的表示
  • 整数
  • 浮点数
  • 复数

与数学中整数的概念一致，可正可负，没有取值范围限制，如pow(2, pow(2, 15))表示很大很大的整数

整数有4种常见的进制表示形式：

与数学中实数一致，取值范围(−10308到10308-10^{308}到10^{308}−10308到10308)和小数精度(10−1610^{-16}10−16)受限制

浮点数间的运算存在不确定尾数，但这不是bug，所有的编程语言都存在该问题，这是十进制和二进制小数无法一一对应导致的，如：

0.1+0.3 = 0.4，但0.1+0.2 = 0.30000000000000004

考虑到不确定尾数，浮点数运算或比较通常用round(x, d)函数来辅助

另外，浮点数也可以用科学记数法来表示，如1.23e-4表示0.000123

字符串类型与操作

字符串类型的表示

字符串即单引号或双引号括起来的有序字符序列，支持索引和切片，有2类4种表示方法：

  • 一对单引号 / 双引号
  • 一对三单引号 / 三双引号

索引格式：<字符串>[M]

切片格式：<字符串>[M:N:K]，M省略表示至开头，N省略表示至结尾，K代表步长

字符串的特殊字符–转义符

转义符可以表达字符的本意

"这里有个双引号(\")" 结果为 这里有个双引号(")

转义符也可以形成一些组合，表达一些不可打印的含义

"\b"回退 "\n"换行(光标移动到下行首) "\r"回车(光标移动到本行首)

字符串操作符
操作符及使用 描述
x + y 连接两个字符串x和y
n * x 或 x*n 复制n次字符x
x in s 如果x是s的子串，返回True，否则返回False
字符串操作函数
函数及使用 描述
len(x) 长度，返回字符串x的长度
len("一二三456")结果为6
str(x) 任意类型x所对应的字符串形式
str(1.23)结果为"1.23" str([1,2])结果为"[1,2]"
hex(x)或oct(x) 整数x的十六进制或八进制小写形式字符串
hex(425)结果为"0x1a9" oct(425)结果为"0o651"
chr(u) u为Unicode编码，返回其对应的字符
ord(x) x为字符，返回其对应的Unicode编码

字符串操作方法

“方法”是面向对象编程中的一个专有名词
  • “方法”特指<a>.<b>()风格中的函数<b>()
  • 方法本身是函数，但与<a>有关，<a>.<b>()风格使用
  • 字符串及变量也是<a>，存在一些方法
方法及使用 描述
str.lower()或str.upper() 返回字符串的副本，全部字符小写/大写
"AbCdEfGk".lower() 结果为 'abcdefgk'
str.split(sep=None) 返回一个列表，由str根据被sep分隔的部分组成
"A,B,C".split(",") 结果为 ['A', 'B', 'C'] #主要用于拆分元素
str.count(sub) 返回子串sub在str中出现的次数
"an apple a day".count("a") 结果为 4
str.replace(old, new) 返回字符串str副本，所有old子串被替换为new
"黑化肥挥发会发灰".replace("发", "fa") 结果为 '黑化肥挥fa会fa灰'
str.center(width[,fillchar]) 字符串根据width居中，fillchar可选
"python".center(20,'=') 结果为 "=======python======="
str.strip(chars) 从str中去掉在其左侧和右侧chars中列出的字符
"= python= ".strip(" =np") 结果为 'ytho'
str.join(iter) 在iter变量除最后一个元素外每个元素后增加一个str
",".join("12345") 结果为 '1,2,3,4,5' #主要用于连接元素

字符串格式化使用.format()方法，用法为：<带有{槽}的字符串>.format(<逗号分隔的参数>)
{槽}内部对格式化的控制方式如下表：

> 右对齐
^ 居中对齐 		槽设定的输出宽度 千分位 浮点数小数位数或字符串最大输出长度 整数类型b,c,d,o,x,X
浮点数类型e,E,f,%

通过Time库练习数字类型和字符串类型的操作

为了避免重复造轮子，Python和第三方封装了很多库供我们调用，其中time库是Python中处理时间的标准库。该库中的函数可以分成三种：

三种 函数 描述
时间获取 time() 获取当前时间戳，即计算机内部时间值，浮点数
>>> time.time() 结果为 1611404718.4370959
gmtime() 时间戳转为计算机可处理的时间格式，UTC时间，不传参则返回当前UTC时间
>>> time.gmtime(1600017509.0) 结果为 time.struct_time(tm_year=2020, tm_mon=9, tm_mday=13, tm_hour=17, tm_min=18, tm_sec=29, tm_wday=6, tm_yday=257, tm_isdst=0)
localtime() 时间戳转为计算机可处理的时间格式，local时间，不传参则返回当前local时间
>>> time.localtime(1600017509.0) 结果为 time.struct_time(tm_year=2020, tm_mon=9, tm_mday=13, tm_hour=21, tm_min=18, tm_sec=29, tm_wday=6, tm_yday=257, tm_isdst=0)
ctime() 时间戳转为人类可识别的时间字符串，不传参则返回当前本地时间
>>> time.ctime() 结果为 'Sun Sep 13 21:18:29 2020'
asctime() 将计算机可处理的时间格式转化为人类可识别的字符串
>>> time.asctime(time.localtime())+"="+time.ctime() 结果为 'Sun Sep 13 21:18:29 2020=Sun Sep 13 21:18:29 2020'
mktime() 将计算机可处理的时间格式转化为时间戳
>>> time.mktime(time.localtime()) 结果为 1600017509.0
时间格式化 strftime(format, t) format是格式化模版字符串，用来自定义输出效果，t是计算机可处理时间格式
>>> time.strftime("%Y-%m-%d") 结果为 '2020-09-13'
strptime(string, format) string是字符串形式的时间，format是格式化模版字符串用来定义输入效果
>>> time.strptime("2020-09-13 12:12:12", "%Y-%m-%d %H:%M:%S") 结果为 time.struct_time(tm_year=2020, tm_mon=9, tm_mday=13, tm_hour=12, tm_min=12, tm_sec=12, tm_wday=6, tm_yday=257, tm_isdst=-1)
程序计时 perf_counter() 返回一个CPU级别的精确时间计数值，单位为秒，由于这个计数值起点不确定，连续调用差值才有意义
>>> start=time.perf_counter()
>>> end=time.perf_counter()
>>> end-start
11.350346330999855
sleep(s) 让程序休眠s秒，s可以是浮点数
>>> def wait(s):
... start=time.perf_counter()
... time.sleep(s)
... end=time.perf_counter()
... print("休眠{:.2f}秒结束".format(s))
>>> wait(3)
休眠3.00秒结束
